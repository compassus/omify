# Omify

om.next-ify plain React components.

## Installation

Leiningen dependency information:

```clojure
[omify "0.1.0"]
```

Maven dependency information:

```xml
<dependency>
  <groupId>omify</groupId>
  <artifactId>omify</artifactId>
  <version>0.1.0</version>
</dependency>
```
